# PharoProject


## Génération documentation exemple

Pour tester le 2nd exercice, il suffit d'ouvrir le Transcript et lancer la méthode de classe sur le nom du paquet voulu :

```Pharo
Transcript open.
PharoDoc generationDocClasses: 'PackageName'.
```
